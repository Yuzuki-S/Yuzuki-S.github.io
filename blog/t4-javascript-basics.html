JavaScript Basics

How does JavaScript compare to HTML and CSS?

JavaScript enables us to implement interactive designs/functions whereas HTML structures content of web pages with paragraphs, lists, images, tables etc and CSS decorates websites by styling the color, the size and the position of letters, and setting the background.

Explain control flow and loops using an example process from everyday life, for example 'waking up' or 'brushing your teeth'.

Control flow is like 'Deciding what to cook for dinner'. For example, "If I have pasta and minced beef, I cook bolognese.", "If I have a lot of vegetables, I cook minestrone soup". Loops are like chopping vegetables just like "cut this vegetable until it is in small pieces".

Explain the difference between accessing data from arrays and objects.

Arrays are used when the data is labelled numerically and is ordered whereas objects are used when the data is labelled semantically but unordered.

Explain what functions are and why they are useful.

A function is a group of code created to implement a series of processes when they are called. It is convinient as people do not have to write the same code every time they need it.
