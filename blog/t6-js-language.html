JavaScript Language

Parentheses ()

1) Parentheses are used to group expressions to change the normal order of precedence. e.g. var x (a + b)*c
2) A function, a set of code designed to perform a particular task uses parentheses to hold its parameters. Parentheses include the list of paremeters the function expects. e.g. function functionName(a,b){} When calling a function, parentheses are used to hold the arguements required by the function. e.g. functionName(a,b)
3) If, for and while statement also holds parentheses to include a condition. e.g. if (a===b)

Brackets []

1)Brackets are used to create a new array. e.g. var animals = ["Dog", "Cat"]
2) These elements can be accessed by referreing the index number. eg. animals[0]// Dog

Braces {}

1) Braces are used to create a new empty object. e.g. ver Dog = {name:"John", age:3}
2) They are also required in function to contain code to be executed e.g. function functionName(){code to be executed}
3) For loop use braces to hold the statement to be executed. e.g. for (let i=0;i<10;i++){to be executed}

Single quotes ' '& Double quotes " "

Single quotes and double quotes are both used to define strings. e.g. var name = "My name" The same type of quotes have to be used in string values.
