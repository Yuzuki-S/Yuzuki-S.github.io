Learning the command line and github

- List of new commands I learnt -

pwd shows the pathway to where you currently are
cd change directry - move to where you need to be
ls list all directory
mkdir make a directory
rmdir remove a directory
cp Example cp A B, A=things to copy (file/folder). B=what to name the copied file/folder and location to put it
echo returns into input as output - returns back to me when I give it
cat print the content of a text file
grep search a word in the given inpute
pipe provides output from one command (left of the pipe symbol) as input to another command (right of the pipe symbol)

- Github -

git add . add everything(.) to a repository I created
git commit -m "comment" prepare a project within the local repository with comment ("")
git push push a project to a remote repository
