What is grid based design?

Grid design is one of the design methods that segments a website or page into grids and determines the layout and the size of the elements by combining the grids.

Whats all the hype about responsive webpages?

Responsive web pages allow users to access the websites regardless of the different size screen in its resolution, web browser or device.

What is semantic structure?

Semantic structure defines the structure of the content/element on a web page using HTML tags such as header, image and forms. The name of the html tag clearly defines what it is used for.
