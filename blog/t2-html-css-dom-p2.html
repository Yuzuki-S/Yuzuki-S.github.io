What is grid based design?

Grid design is one of the design methods that segments a website or page into grids and determines the layout and the size of the elements by combining the grids.

Whats all the hype about responsive webpages?

Responsive web pages are a web design technique that allow users to access to the websites regardless of the different size screen in its resolution, web browser or device.

What is semantic structure?
